# Computational Physics using Python

This repository contains Python implementations of basic numerical methods
and physics simulations commonly used in computational physics.

## Topics Covered
- Root finding methods (Bisection, Newton-Raphson)
- Numerical solution of differential equations (Euler method)
- Classical mechanics simulations
- Data visualization using Matplotlib

## Tools Used
- Python
- NumPy
- Matplotlib

## Academic Context
Created as part of learning computational physics and numerical methods.
